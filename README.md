# GE简主义 (GEIdea) - 极简主义桌面工具集

一个基于Electron开发的简约桌面应用程序，集成了日历、待办清单、番茄钟、天气和倒数日等实用工具，专注于提供简洁高效的用户体验。

## 🌟 项目简介

GE简主义是一个轻量级的桌面工具集合，采用现代化的半透明设计风格，为用户提供了日常所需的各种小工具。项目秉承"少即是多"的设计理念，通过简洁的界面和实用的功能，帮助用户更好地管理时间和任务。

## ✨ 功能特性

### 🗓️ 日历组件
- **月度视图展示**：清晰的月历显示，支持月份切换
- **日程管理**：点击日期可添加和管理当日日程
- **事件计数显示**：日期上显示当日日程数量
- **本地存储**：所有日程数据保存在本地，无需网络连接
- **快速操作**：支持日程的添加、删除和刷新功能

### 📝 待办清单
- **任务管理**：创建和管理待办事项
- **日期标记**：为每个任务设置具体日期
- **完成状态**：通过勾选标记任务完成状态
- **数据持久化**：使用localStorage保存任务数据
- **简洁界面**：清晰的任务列表展示

### 🍅 番茄钟
- **专注计时**：基于番茄工作法的时间管理
- **自定义时长**：支持设置工作和休息时间
- **视觉提醒**：直观的倒计时显示
- **效率统计**：记录专注时间和完成次数

### 🌤️ 天气组件
- **实时天气**：显示当前天气状况
- **位置定位**：自动获取用户位置信息
- **简洁展示**：小窗口设计，不占用过多屏幕空间
- **快速查看**：一键查看天气信息

### 📅 倒数日
- **重要日期**：记录和追踪重要日期
- **倒计时显示**：直观显示距离目标日期的天数
- **事件分类**：支持不同类型的事件管理
- **提醒功能**：临近日期时提供提醒

## 🎨 设计特色

### 视觉设计
- **半透明效果**：采用现代化的毛玻璃效果
- **深色主题**：护眼的深色界面设计
- **简约图标**：精心设计的图标系统
- **流畅动画**：细腻的过渡和交互动画

### 用户体验
- **一键启动**：从主界面快速访问所有工具

## 🛠️ 技术架构

### 核心技术
- **Electron 34.1.0**：跨平台桌面应用框架
- **HTML5 + CSS3**：现代化的前端技术栈
- **JavaScript ES6+**：原生JavaScript实现
- **Node.js**：后端运行时环境

### 核心模块

#### 主进程 (main.js)
- **窗口管理**：负责创建和管理各个功能窗口
- **IPC通信**：处理渲染进程与主进程间的通信
- **窗口配置**：设置窗口属性、透明度和位置
- **生命周期管理**：应用的启动、关闭和事件处理

#### 预加载脚本 (preload.js)
- **安全通信**：使用contextBridge实现安全的IPC通信
- **API暴露**：向渲染进程暴露主进程功能
- **权限控制**：限制渲染进程的直接系统访问

#### 渲染进程 (render.js)
- **事件处理**：处理用户界面交互
- **API调用**：调用主进程提供的功能
- **状态管理**：管理组件的显示和隐藏

## 🚀 快速开始

### 环境要求
- Node.js 16.0 或更高版本
- npm 或 yarn 包管理器

### 安装依赖
```bash
npm install
```

### 开发模式运行
```bash
npm start
```

### 构建应用
```bash
npm run build
```

### 打包配置
项目已配置electron-builder，支持生成Windows安装程序：
- 应用ID：com.geidea.own
- 图标文件：geid.png
- 支持x64架构
- 自定义安装路径

## 💻 开发指南

### 代码规范
- 使用语义化的HTML标签
- CSS采用BEM命名规范
- JavaScript使用ES6+语法
- 遵循DRY（Don't Repeat Yourself）原则

### 性能优化
- 使用事件委托减少事件监听器
- 采用防抖和节流优化频繁操作
- 合理使用CSS动画代替JavaScript动画
- 优化图片资源和字体加载

### 安全考虑
- 启用contextIsolation隔离渲染进程
- 禁用remote模块直接访问
- 使用预加载脚本控制API暴露
- 验证所有用户输入

## 🎯 使用说明

### 主界面操作
1. 启动应用后显示主界面
2. 点击对应图标打开功能组件
3. 每个组件都是独立的可移动窗口
4. 支持同时打开多个组件

### 日历使用
1. 点击日历图标打开日历窗口
2. 使用左右箭头切换月份
3. 点击具体日期查看和添加日程
4. 日程数量会显示在日期上

### 待办清单使用
1. 点击待办清单图标打开窗口
2. 点击"DOIT"按钮显示添加界面
3. 输入日期和事件内容
4. 点击添加按钮创建任务
5. 点击勾选按钮标记完成

### 组件管理
- 日历和天气组件默认定位到右上角
- 所有组件支持自由拖动
- 组件窗口支持透明度调节
- 点击组件图标可重新聚焦已打开的窗口

## 🔧 配置选项

### 窗口配置
- **透明度**：主窗口90%，组件窗口80%
- **背景色**：rgba(29, 29, 29, 0.51)
- **窗口大小**：根据功能需求定制
- **定位策略**：智能屏幕定位

### 样式配置
- **主题色**：深灰色主题
- **强调色**：#e66d1c（橙色）
- **字体**：使用自定义苹方字体
- **图标**：Font Awesome 4.7.0

## 📊 数据存储

### 存储机制
- 使用浏览器localStorage API
- 数据存储在用户本地
- 无需网络连接即可使用
- 支持数据的导入导出

### 数据结构
```javascript
// 日程数据
events-${year}-${month}-${day}: [event1, event2, ...]

// 待办事项数据
todos: [{date: "日期", things: "内容"}, ...]
```
